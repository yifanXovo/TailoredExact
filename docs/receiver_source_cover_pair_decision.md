# Receiver-Source-Cover Pair Decision

Singleton receiver-source-cover remains paper-safe.

Pair/set receiver cover remains diagnostic-only in this round. The unsafe old
form can cut feasible solutions when receiver-set stations exchange bikes
internally. The pair-net formulation is plausible, but the implementation was
not enabled as paper evidence because it still needs a complete feasible-route
projection proof and randomized validation.

Controlled paper rows therefore use only singleton-paper-safe receiver cover.
