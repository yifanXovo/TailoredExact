# S-Bucket Or Split Diagnostic

S-bucket refinement remains diagnostic in this round. The fixed-interval child rows may report `s_range_refinement_enabled`, parent S-domain aliases, and bucket bounds, but paper certificate use requires exact child coverage of the full parent S-domain and a merged ledger proving every bucket. The round-level `s_bucket_coverage_audit.csv` rejects diagnostic buckets if they are marked as certificate-valid or used as parent evidence.
