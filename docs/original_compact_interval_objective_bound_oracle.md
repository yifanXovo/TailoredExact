# Original Compact Interval Objective-Bound Oracle

The objective-bound oracle is an exact compact MIP over one Gini interval. It
uses the same original compact model as the interval cutoff oracle, adds
`gamma_L <= G <= gamma_U`, and minimizes the original weighted objective
`G + lambda * P`.

The oracle supports timeout-derived lower bounds:

- `optimal`: the optimal objective is the interval lower bound;
- `infeasible`: the interval is empty in the original compact model;
- `time limit`: CPLEX's finite best bound is a valid interval lower bound;
- feasible incumbent below the current UB: the incumbent is verifier-gated and
  can only update UB, requiring a full frontier restart.

This oracle is not an incumbent source and it does not import route-pool or
archive evidence. It contributes only lower-bound evidence generated by the
same original fixed-interval compact model.
