\section{Scope}

The recommended exact algorithm is a tailored exact portfolio for the original EBRP objective
\[
\min\; G+\lambda P.
\]
The portfolio contains pure full Gini-frontier route-load BPC as the main decomposition method and strengthened compact branch-and-cut as an auxiliary exact fallback. Plain CPLEX is the benchmark. Positive-gap results are never reported as optimal.

\section{Method}

The BPC frontier covers at least $G\in[0,UB]$. For each interval $[\gamma_L,\gamma_U]$, it uses
\[
G\le \gamma_U \Longleftrightarrow H\le V\gamma_U S,
\]
plus $G\ge\gamma_L$. Columns are complete route-load columns. A restricted-pool result is not a certificate; an interval must be closed by exact pricing or bound-fathomed by a valid lower bound.

The implementation includes exact pricing, Ryan-Foster/co-route branching, subset-row cuts, operation-time conservation, final-inventory/Gini relaxations, complete route-mask duration/load relaxations for $V\le 12$, BPC-owned incumbents, and station-operation cuts
\[
p_i+d_i\le U_i v_i,\qquad p_i+d_i\ge v_i.
\]

\section{Certificate Protocol}

BPC certifies the original problem only when \texttt{status=optimal}, \texttt{gap=0}, $LB=UB=\text{objective}$, the verifier passes, and the full frontier ledger has zero unresolved intervals, zero invalid bound intervals, and zero top-level open nodes. Compact fallback certificates are exact original-problem certificates when their compact MIP gap closes, but they are not BPC certificates.

\section{Interpretation}

Pure BPC certifies V10 M2 average, V10 M1 average, and V12 M1 average. The strengthened compact fallback certifies V10 M2 low and is also faster on M1-style samples, but it is not BPC. Plain CPLEX remains the benchmark and is faster on V10 M1 and V12 M1.

For V10 M2 average, the correct statement is: BPC certified in 589.00 seconds; plain CPLEX did not certify within 1200.15 seconds, final gap 0.0245397. For V10 M2 low, the portfolio certified by compact fallback in 2138.81 seconds; plain CPLEX did not certify within 300.09 seconds, final gap 0.0379116.
